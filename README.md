## bf.py

Brainfuck interpreter written in Python.

```
python bf.py
```


In bf.py file:
```
execute(bfcode, input[optional])
```

### Example
```
print(execute("++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.", ""))


# Output: Hello World!
```
